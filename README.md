# 🎬 Netflix Data Analysis with Python

A complete **Exploratory Data Analysis (EDA)** project on the **Netflix Titles Dataset** using **Python, Pandas, NumPy, and Matplotlib**. This project focuses on cleaning real-world data, analyzing trends, extracting insights, and creating professional visualizations.

## 📌 Project Overview

This project analyzes the Netflix Titles dataset to understand:

* Content distribution
* Release trends
* Country-wise production
* Popular genres
* Content ratings
* Director and actor frequency
* Monthly content additions
* Movie duration patterns
* Release year distribution
* Content type comparison

The project includes **12 analytical insights** supported by **12 visualizations**.

---

## 📊 Dataset Information

* **Dataset:** Netflix Titles
* **Total Records:** ~8,800+
* **Columns:** 12
* **Missing Values Handled:** 5 Columns

  * Director
  * Cast
  * Country
  * Date Added
  * Rating
* **Tools Used:**

  * Python 3
  * Pandas
  * Matplotlib

## 🧹 Data Cleaning

The dataset was cleaned before analysis by:

* Filling missing values in **Director**
* Filling missing values in **Cast**
* Filling missing values in **Country**
* Filling missing values in **Rating**
* Removing extra spaces from **Date Added**
* Converting **Date Added** into DateTime format
* Saving the cleaned dataset
